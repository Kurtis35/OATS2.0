GOOGLE SHEETS BOOKING INTEGRATION

Transport bookings must be stored in Google Sheets using a Google Apps Script Web App endpoint.

Use this endpoint:

https://script.google.com/macros/s/AKfycbyKgq_pUO2wgYmsZxLDWE81v3MIaLg4exyGh_x7CNY/dev

When the booking form is submitted:

1 Prevent page reload
2 Collect all form data
3 Send data to the Apps Script endpoint using fetch

Example structure:

fetch(SCRIPT_URL, {
method: "POST",
body: JSON.stringify(data)
})

After successful submission:

Display confirmation message:

"Thank you. Your transport booking has been received."

Then reset the form.

Admins must be able to view all bookings directly inside Google Sheets.
